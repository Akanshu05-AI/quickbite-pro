# QuickBite Pro 
**Smart Mess Management System**

QuickBite Pro is a lightweight, front-end solution for managing student mess attendance using QR-based tokens.

###  Features
* **Dynamic Menu:** Updates based on the time of day.
* **QR Token Generation:** Instant digital passes for students.
* **Admin Dashboard:** Hidden panel (Triple-click header) for real-time analytics.
* **Security Logs:** Prevents double-entry for the same meal session.
* **Data Export:** Export attendance logs to CSV.

###  Tech Stack
* HTML5 / CSS (Flexbox/Grid)
* Vanilla JavaScript
* QRCode.js Library
